## 前言

我是刘小满，中南民族大学计算机专业大四学生。从大二开始自学前端，到现在两年了。这篇文章回顾一下整个过程，也写给正在学前端的学弟学妹们参考。

## 2023.09 — 入门

大二上学期，第一门 Web 课让我知道了 HTML 和 CSS 的存在。课后自己找 MDN 文档看，写了人生第一个网页——一个丑到不忍直视的个人主页。

但那种"写几行代码就能在浏览器里看到效果"的感觉，真的很上头。

## 2024.03 — 入坑 Vue

大二下学期开始学 Vue。刚接触组件化的时候概念很抽象——为什么要拆组件？后来做项目多了才理解：**组件就是乐高积木，拼起来比捏泥巴快多了。**

第一个完整的 Vue 项目是一个待办事项应用，实现了增删改查、本地存储。功能简单，但写完那一刻特别有成就感。

## 2024.09 — 升级 Vue 3

大三上学期，决定从 Vue 2 切换到 Vue 3。Composition API 一开始很不习惯，总觉得写法不如 Options API 直观。

转折点是我写了一个表单验证的 composable：

```typescript
function useForm(rules) {
  const errors = ref({})
  function validate(field, value) { ... }
  return { errors, validate }
}
```

突然发现逻辑复用变得这么自然，从此彻底接受了 Composition API。

## 2025.03 — 工程化

大三下学期开始关注工程化：TypeScript 类型系统、Pinia 状态管理、Vite 构建优化。不再是"能跑就行"，开始追求代码质量和开发体验。

## 2025.09 — 秋招

大四上学期全力准备秋招。刷题、写项目、整理博客。这个个人博客网站也是这段时间做的——用来展示作品和沉淀技术文章。

## 给学弟学妹的建议

1. **基础很重要**——JS 基础不牢，学框架会很痛苦
2. **多写项目**——看一百遍文档不如自己动手做一个
3. **别怕踩坑**——每个 bug 都是学习机会
4. **写博客记录**——输出是最好的学习方式
5. **坚持**——前端东西多，但不用急，一步一步来

## 最后

两年时间不长，但足以从零基础到能独立完成项目。前端这条路，越走越有意思。
