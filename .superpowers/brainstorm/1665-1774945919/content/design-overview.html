设计特色

✦ 毛玻璃卡片

所有内容区块使用 backdrop-filter 模糊效果，背景薄荷渐变透出

✦ 滚动动画

Intersection Observer 驱动的渐入动画，各模块优雅显现

✦ 暗色模式

一键切换明/暗主题，暗色下薄荷绿更突出

✦ 响应式

移动端自适应，汉堡菜单，卡片单列堆叠

数据方案

博客文章和项目数据用本地 JSON 文件存储，Markdown 渲染用 markdown-it。无需后端，纯静态即可部署到 GitHub Pages / Vercel。
